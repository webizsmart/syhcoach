# Song Young-ho | Global Badminton Coach

## 프로젝트 소개
한국 장애인 배드민턴의 개척자이자 세계적인 지도자인 **송영호 감독**의 공식 홍보 웹사이트입니다.

## 기술 스택
- **HTML5**: 시맨틱 마크업 설계
- **Vanilla CSS**: 다크 네이비 + 골드 팔레트를 활용한 커스텀 디자인 시스템 (Variables, Flexbox/Grid, Animations)
- **Vanilla JavaScript**: 다국어 처리(한국어/캄보디아어), 스크롤 애니메이션, 카운터 인터랙션
- **Google Fonts & FontAwesome**: Noto Sans KR, Bebas Neue, Outfit 폰트 적용 및 아이콘 활용

## 주요 기능
- **바이링구얼 시스템**: 우측 상단 언어 토글(KO/KM)을 통해 페이지 내용 전체가 한국어와 캄보디아어(Khmer)로 즉시 전환됨.
- **인터랙티브 UI**: 영웅(Hero) 영역 글리치(Glitch) 이펙트, 스크롤 진입 시 숫자 카운터 애니메이션, 부드러운 스크롤 앵커(anchor) 네비게이션.
- **반응형 웹**: 모바일 기기를 고려한 햄버거 메뉴 및 그리드 재정렬 구현.

## 파일 구조
- `index.html`: 메인 페이지 마크업
- `style.css`: 전체 스타일시트 및 반응형 룰셋
- `script.js`: 다국어 번역 데이터(translations 객체) 매핑 및 UI 인터랙션 스크립트
- `README.md`: 프로젝트 개요 설명
